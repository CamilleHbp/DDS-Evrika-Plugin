Krita DDS Evrika Plugin - User Manual

Introduction

The DDS Evrika Plugin is an extension designed for Krita, enabling users to import and export DDS image files using ImageMagick. DDS (DirectDraw Surface) is a popular format for storing textures and images optimized for real-time rendering, often used in games and 3D applications.

Installation

  1. Download the plugin files.
  2. Copy the plugin directory to Krita's plugin folder:
    • Windows: C:\Users\\AppData\Roaming\krita\pykrita
    • Linux: ~/.local/share/krita/pykrita
  3. Restart Krita.
  4. Go to the "Tools > Scripts" menu and enable DDS Evrika Plugin.

Usage

Importing DDS Files

  1. Go to Tools > Scripts > Import DDS.
  2. Select the DDS file you want to import.
  3. Select the target format for conversion (TGA, PNG, etc.).
  4. The image will open within Krita after successful conversion.

Exporting DDS Files

  1. Go to Tools > Scripts > Export to DDS.
  2. Select the compression format for the DDS file (DXT1, DXT3, DXT5).
  3. The image will be exported and saved as a DDS file.

Troubleshooting

If you encounter issues with the conversion process:

  • Ensure ImageMagick is properly installed in the "resources" folder.
